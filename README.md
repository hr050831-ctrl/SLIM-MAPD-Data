# Supplementary Data for "Sparse Lightweight Integration for Lifelong MAPD"

This repository contains the supplementary experimental data, extended ablation tables, and raw execution logs for our submission to the IEEE Transactions on Automation Science and Engineering (T-ASE).

## Repository Structure
- `/data`: Contains the raw execution logs, memory profiling data, and the 870k+ safe-interval gap distributions referenced in the empirical bounds validation.
- `/extended_tables`: Full versions of the ablation study tables (including standard deviations and additional metrics) omitted from the main manuscript due to space constraints.
- `/figures`: High-resolution performance scaling plots and background thread heatmaps.

## Source Code Availability
**Note:** To protect the integrity of the review process and prevent scooping before formal publication, the core C++ source code of the SLIM framework and the FSH-A* planner is currently withheld. 

The complete codebase will be made fully open-source and publicly available in this repository immediately upon the acceptance of the manuscript.
